# Spring - Vue.js Prototype
A prototype, implemented with Spring Boot and Vue.js.

### The project has two modules
- Spring backend
- Vue.js frontend

The frontend is built with:
- node v16.18.0
- vue/cli@5.0.8
- npm@8.19.2

### Backend
```shell
./gradlew bootRun
./gradlew bootWar
```

### Frontend
```shell
npm install
npm run serve
```
