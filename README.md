# Spring - Vue.js Prototype
A prototype, implemented with Spring Boot and Vue.js.

### The project has two modules
- Spring backend
- Vue.js frontend

> The frontend was built with Node v16.18.0 and Vue CLI v5.0.8.
>
### Backend
```shell
./gradlew bootRun
./gradlew bootWar
```

### Frontend
```shell
npm install
npm run serve
```
